Description:
------------
This calculator implementation consists of two HTML pages:
1. calculator.html - The main calculator form interface
2. calculator-result.html - The results display page

Features:
---------
- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Advanced operations (Power, Square Root, Percentage)
- Memory functions (Store, Recall, Clear)
- Form-based calculation interface using HTML5 form elements
- Semantic HTML5 structure with proper fieldset and legend elements

Implementation Notes:
---------------------
- The calculator uses HTML5 form elements (input, button, fieldset, legend)
- Form validation is implemented using HTML5 required attributes
- The form submits to calculator-result.html using GET method
- All form elements are properly labeled for accessibility
- Semantic HTML5 structure is used throughout

File Structure:
---------------
Part4_Calculator/
  - calculator.html
  - calculator-result.html
  - README.txt (this file)

Usage:
------
1. Open calculator.html in a web browser
2. Enter two numbers in the input fields
3. Select an operation (basic or advanced)
4. Click "Calculate" to submit the form
5. View results on calculator-result.html

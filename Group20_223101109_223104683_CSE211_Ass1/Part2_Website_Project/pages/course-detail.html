Course Overview

Description

This course is designed for absolute beginners. We start with the basics of how the web works and move quickly into writing your own code. By the end of the 8 weeks, you will have built a personal portfolio website to showcase your new skills.

What You Will Learn

  1. Structure web pages using semantic HTML5.
  2. Style websites with CSS3, Flexbox, and Grid.
  3. Add interactivity with modern JavaScript (ES6+).
  4. Deploy your projects to the live web.

Prerequisites

  • Basic computer literacy.
  • No prior coding experience required.
  • A computer with internet access.

Level Badge: Beginner

Time Commitment: 8-10 hours/week

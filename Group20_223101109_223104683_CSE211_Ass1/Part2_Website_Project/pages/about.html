About CSE211 Web Programming

Welcome in CSE211 Web Programming. This course is designed to introduce students to the fundamental concepts of web development. From structuring content with HTML5 to styling with CSS and adding interactivity, this course provides the essential building blocks for the modern web.

Course Objectives

  1. Understand the architecture of the World Wide Web and HTTP protocol.
  2. Master the use of Semantic HTML5 for accessible and structured content.
  3. Analyze web performance metrics including Core Web Vitals.
  4. Develop responsive layouts that work on mobile and desktop devices.
  5. Implement client-side form validation and user input handling.

Topics Covered

  • Web Design & Architecture: UI/UX principles, prototyping, and layout planning.
  • HTML5 Semantics: Structuring content with proper tags for better accessibility.
  • CSS Styling: Mastering selectors, Flexbox, and responsive design techniques.
  • JavaScript Fundamentals: Adding interactivity and DOM manipulation.
  • Backend Basics: Introduction to server-side logic and database connectivity.
  • Web Security: Understanding common vulnerabilities and secure coding practices.
